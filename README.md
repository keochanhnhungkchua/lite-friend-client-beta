# Lite Friend 
    * code phần giao diện của Facebook đồng thời thêm bớt một số tính năng khác 
        * chat sử dụng socketIo
        * upload story
        * post bài :like comment
        * ...
## Backend : sử dụng react
    * socketIo chat
    * UI: material-ui
    * State: Recoil
## Fontend: sử dụng NodeJs 
    * DB: monggoDB
    * Chat: socketIo
